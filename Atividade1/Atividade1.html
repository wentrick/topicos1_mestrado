ITEM (a): Função de densidade correspondente à CDF do slide 15

A CDF da distribuição de Morgenstern bivariada com marginais Gamma é:

\[ F_{X,Y}(x,y) = F_X(x)\,F_Y(y)\,\big[1 + \rho\{1 - F_X(x)\}\{1 - F_Y(y)\}\big], \]

onde \(F_X \sim \text{Gamma}(\alpha,\theta)\), \(F_Y \sim \text{Gamma}(\beta,\theta)\) e \(-1 \leqslant \rho \leqslant 1\).

Queremos obter a PDF bivariada:

\[ f_{X,Y}(x,y) = \frac{\partial^2 F_{X,Y}}{\partial x\,\partial y}(x,y). \]

Para simplificar, denotamos \(u = F_X(x)\) e \(v = F_Y(y)\), de modo que:

\[ F = u\,v\,\big[1 + \rho(1-u)(1-v)\big]. \]

Passo 1: Expansão do produto

Expandindo os parênteses:

\[ \begin{aligned} F &= u\,v\,\big[1 + \rho(1 - u - v + uv)\big] \\ &= uv + \rho\,uv\,(1 - u - v + uv) \\ &= uv + \rho\big(uv - u^2v - uv^2 + u^2v^2\big). \end{aligned} \]

Passo 2: Primeira derivada \(\frac{\partial F}{\partial u}\)

Derivando termo a termo em relação a \(u\):

\[ \begin{aligned} \frac{\partial F}{\partial u} &= v + \rho\big(v - 2uv - v^2 + 2uv^2\big) \\ &= v + \rho\,v\big(1 - 2u - v + 2uv\big). \end{aligned} \]

Observe que a expressão entre parênteses se fatoriza:

\[ 1 - 2u - v + 2uv = (1-2u)(1-v), \]

Logo:

\[ \boxed{\frac{\partial F}{\partial u} = v\,\big[1 + \rho(1-2u)(1-v)\big]}. \]

Passo 3: Derivada cruzada \(\frac{\partial^2 F}{\partial u\,\partial v}\)

Derivamos o resultado do Passo 2 em relação a \(v\). Como temos um produto de \(v\) por uma função de \(v\), aplicamos a regra do produto:

\[ \begin{aligned} \frac{\partial^2 F}{\partial u\,\partial v} &= \frac{\partial}{\partial v}\Big\{v\,\big[1 + \rho(1-2u)(1-v)\big]\Big\} \\[6pt] &= 1 \cdot \big[1 + \rho(1-2u)(1-v)\big] \;+\; v \cdot \big[\rho(1-2u)\cdot(-1)\big] \\[6pt] &= 1 + \rho(1-2u)(1-v) - \rho\,v(1-2u). \end{aligned} \]

Colocando \(\rho(1-2u)\) em evidência nos dois últimos termos:

\[ \begin{aligned} \frac{\partial^2 F}{\partial u\,\partial v} &= 1 + \rho(1-2u)\big[(1-v) - v\big] \\ &= 1 + \rho(1-2u)(1 - 2v). \end{aligned} \]

Portanto:

\[ \boxed{\frac{\partial^2 F}{\partial u\,\partial v} = 1 + \rho(1-2u)(1-2v)}. \]

Passo 4: Regra da cadeia (retorno às variáveis \(x\) e \(y\))

Lembrando que \(u = F_X(x)\) e \(v = F_Y(y)\), a regra da cadeia nos dá:

\[ \frac{\partial^2 F}{\partial x\,\partial y} = \frac{\partial^2 F}{\partial u\,\partial v} \cdot \frac{\partial u}{\partial x} \cdot \frac{\partial v}{\partial y}. \]

Como \(u = F_X(x)\), temos \(\frac{\partial u}{\partial x} = f_X(x)\) (a PDF marginal de \(X\)). Analogamente, \(\frac{\partial v}{\partial y} = f_Y(y)\). Substituindo:

\[ \boxed{f_{X,Y}(x,y) = f_X(x)\,f_Y(y)\,\big[1 + \rho(1 - 2F_X(x))(1 - 2F_Y(y))\big]}, \]

onde as densidades marginais Gamma são:

\[ f_X(x) = \frac{x^{\alpha-1}\,e^{-x/\theta}}{\theta^\alpha\,\Gamma(\alpha)}, \qquad f_Y(y) = \frac{y^{\beta-1}\,e^{-y/\theta}}{\theta^\beta\,\Gamma(\beta)}. \]

Verificação

Tomando \(\rho = 0\):

\[ f_{X,Y}(x,y) = f_X(x)\,f_Y(y)\,\big[1 + 0\big] = f_X(x)\,f_Y(y), \]

que é o produto das marginais, confirmando a independência quando \(\rho = 0\).

# --- Funções do modelo ---

# CDF bivariada
cdf_morgenstern <- function(x, y, alpha, beta, theta, rho) {
  u <- pgamma(x, shape = alpha, rate = 1/theta)
  v <- pgamma(y, shape = beta, rate = 1/theta)
  u * v * (1 + rho * (1 - u) * (1 - v))
}

# PDF bivariada
pdf_morgenstern <- function(x, y, alpha, beta, theta, rho) {
  fx <- dgamma(x, shape = alpha, rate = 1/theta)
  fy <- dgamma(y, shape = beta, rate = 1/theta)
  Fx <- pgamma(x, shape = alpha, rate = 1/theta)
  Fy <- pgamma(y, shape = beta, rate = 1/theta)
  fx * fy * (1 + rho * (1 - 2 * Fx) * (1 - 2 * Fy))
}

ITEM (b): Estimadores de Máxima Verossimilhança (EMV)

Pela PDF que derivamos no item (a), cada observação contribui com:

\[ \log f_{X,Y}(x_i, y_i) = \log\Big[f_X(x_i)\,f_Y(y_i)\,\big(1 + \rho\,h_i\big)\Big]. \]

Usando a propriedade \(\log(a \cdot b \cdot c) = \log a + \log b + \log c\):

\[ \log f_{X,Y}(x_i, y_i) = \underbrace{\log f_X(x_i)}_{\text{parte (A)}} + \underbrace{\log f_Y(y_i)}_{\text{parte (B)}} + \underbrace{\log(1 + \rho\,h_i)}_{\text{parte (C)}}, \]

onde \(h_i = (1 - 2F_X(x_i))(1 - 2F_Y(y_i))\).

Vamos expandir cada parte separadamente.

Parte (A) — Expandindo \(\log f_X(x_i)\)

A densidade da \(\text{Gamma}(\alpha, \theta)\) é:

\[ f_X(x) = \frac{1}{\theta^\alpha\,\Gamma(\alpha)}\,x^{\alpha-1}\,e^{-x/\theta}. \]

Aplicando o \(\log\) e usando \(\log(a \cdot b \cdot c) = \log a + \log b + \log c\):

\[ \log f_X(x_i) = \log\left(\frac{1}{\theta^\alpha\,\Gamma(\alpha)}\right) + \log\left(x_i^{\alpha-1}\right) + \log\left(e^{-x_i/\theta}\right). \]

Simplificando cada termo:

\[ \begin{aligned} \log\left(\frac{1}{\theta^\alpha\,\Gamma(\alpha)}\right) &= -\log(\theta^\alpha) - \log\Gamma(\alpha) = -\alpha\log\theta - \log\Gamma(\alpha), \\[6pt] \log\left(x_i^{\alpha-1}\right) &= (\alpha-1)\log x_i, \\[6pt] \log\left(e^{-x_i/\theta}\right) &= -\frac{x_i}{\theta}. \end{aligned} \]

Juntando os três termos:

\[ \boxed{\log f_X(x_i) = (\alpha-1)\log x_i - \frac{x_i}{\theta} - \alpha\log\theta - \log\Gamma(\alpha)}. \]

Parte (B) — Expandindo \(\log f_Y(y_i)\)

O procedimento é idêntico ao da Parte (A), trocando \(\alpha \to \beta\) e \(x_i \to y_i\). A densidade marginal é:

\[ f_Y(y) = \frac{1}{\theta^\beta\,\Gamma(\beta)}\,y^{\beta-1}\,e^{-y/\theta}, \]

e portanto:

\[ \boxed{\log f_Y(y_i) = (\beta-1)\log y_i - \frac{y_i}{\theta} - \beta\log\theta - \log\Gamma(\beta)}. \]

Parte (C) — O termo da cópula

Este termo fica como está:

\[ \log(1 + \rho\,h_i), \]

onde \(h_i\) depende das CDFs Gamma avaliadas nos dados:

\[ h_i = \big(1 - 2\,F_X(x_i;\alpha,\theta)\big)\,\big(1 - 2\,F_Y(y_i;\beta,\theta)\big), \]

sendo \(F_X(x_i;\alpha,\theta) = \frac{1}{\Gamma(\alpha)}\int_0^{x_i/\theta} t^{\alpha-1}e^{-t}\,dt\) a função Gamma incompleta regularizada (a CDF da Gamma).

Note que este é o termo que acopla todos os parâmetros: \(\alpha\), \(\beta\), \(\theta\) e \(\rho\) aparecem juntos dentro de \(h_i\).

Log-verossimilhança completa

Somando as partes (A), (B) e (C) para todas as \(n\) observações:

\[ \ell(\alpha,\beta,\theta,\rho) = \sum_{i=1}^{n}\bigg[(\alpha-1)\log x_i + (\beta-1)\log y_i - \frac{x_i + y_i}{\theta} - (\alpha+\beta)\log\theta - \log\Gamma(\alpha) - \log\Gamma(\beta) + \log\big(1 + \rho\,h_i\big)\bigg]. \]

Separando os termos que não dependem de \(i\) (constantes ao longo da soma), podemos reescrever como:

\[ \begin{aligned} \ell = \;&(\alpha-1)\underbrace{\sum_{i=1}^n \log x_i}_{:=\, S_x} + (\beta-1)\underbrace{\sum_{i=1}^n \log y_i}_{:=\, S_y} - \frac{1}{\theta}\underbrace{\sum_{i=1}^n (x_i + y_i)}_{=\, n(\bar{x}+\bar{y})} \\[8pt] &- n(\alpha+\beta)\log\theta - n\log\Gamma(\alpha) - n\log\Gamma(\beta) + \sum_{i=1}^n \log(1+\rho\,h_i), \end{aligned} \]

onde definimos as estatísticas suficientes:

  • \(S_x = \sum_{i=1}^n \log x_i\) e \(S_y = \sum_{i=1}^n \log y_i\) (somas dos log-dados),
  • \(\bar{x}\) e \(\bar{y}\) (médias amostrais).
# Função de log-verossimilhança negativa
neg_loglik <- function(par, x, y) {
  alpha <- par[1]
  beta  <- par[2]
  theta <- par[3]
  rho   <- par[4]
  
  # Restrições de domínio
  if (alpha <= 0 || beta <= 0 || theta <= 0 || abs(rho) > 1) return(1e10)
  
  fx <- dgamma(x, shape = alpha, rate = 1/theta, log = TRUE)
  fy <- dgamma(y, shape = beta, rate = 1/theta, log = TRUE)
  Fx <- pgamma(x, shape = alpha, rate = 1/theta)
  Fy <- pgamma(y, shape = beta, rate = 1/theta)
  
  copula_term <- 1 + rho * (1 - 2 * Fx) * (1 - 2 * Fy)
  
  # Verificar se o termo da cópula é positivo (condição de densidade)
  if (any(copula_term <= 0)) return(1e10)
  
  ll <- sum(fx + fy + log(copula_term))
  return(-ll)
}

# Reparametrização para otimização irrestrita (método BFGS)
# alpha = exp(phi1), beta = exp(phi2), theta = exp(phi3), rho = tanh(phi4)
neg_loglik_repar <- function(phi, x, y) {
  par <- c(exp(phi[1]), exp(phi[2]), exp(phi[3]), tanh(phi[4]))
  neg_loglik(par, x, y)
}

# Função para estimar com múltiplas estratégias
fit_morgenstern <- function(x, y, verbose = TRUE) {
  
  # Valores iniciais baseados nos momentos marginais
  mean_x <- mean(x); var_x <- var(x)
  mean_y <- mean(y); var_y <- var(y)
  
  theta0 <- (var_x / mean_x + var_y / mean_y) / 2
  alpha0 <- mean_x / theta0
  beta0  <- mean_y / theta0
  rho0   <- 0.1 * sign(cor(x, y))
  
  par0 <- c(alpha0, beta0, theta0, rho0)
  
  results <- list()
  
  # --- Estratégia 1: Nelder-Mead ---
  t1 <- system.time({
    fit1 <- tryCatch(
      optim(par0, neg_loglik, x = x, y = y, method = "Nelder-Mead",
            control = list(maxit = 10000, reltol = 1e-10)),
      error = function(e) NULL
    )
  })
  if (!is.null(fit1)) {
    results[["Nelder-Mead"]] <- list(
      par = fit1$par, loglik = -fit1$value,
      convergence = fit1$convergence, time = t1["elapsed"]
    )
  }
  
  # --- Estratégia 2: L-BFGS-B (com box constraints) ---
  t2 <- system.time({
    fit2 <- tryCatch(
      optim(par0, neg_loglik, x = x, y = y, method = "L-BFGS-B",
            lower = c(1e-4, 1e-4, 1e-4, -0.999),
            upper = c(Inf, Inf, Inf, 0.999),
            control = list(maxit = 10000)),
      error = function(e) NULL
    )
  })
  if (!is.null(fit2)) {
    results[["L-BFGS-B"]] <- list(
      par = fit2$par, loglik = -fit2$value,
      convergence = fit2$convergence, time = t2["elapsed"]
    )
  }
  
  # --- Estratégia 3: BFGS com reparametrização ---
  phi0 <- c(log(alpha0), log(beta0), log(theta0), atanh(max(-0.99, min(0.99, rho0))))
  
  t3 <- system.time({
    fit3 <- tryCatch(
      optim(phi0, neg_loglik_repar, x = x, y = y, method = "BFGS",
            control = list(maxit = 10000, reltol = 1e-10)),
      error = function(e) NULL
    )
  })
  if (!is.null(fit3)) {
    par3 <- c(exp(fit3$par[1]), exp(fit3$par[2]), exp(fit3$par[3]), tanh(fit3$par[4]))
    results[["BFGS-repar"]] <- list(
      par = par3, loglik = -fit3$value,
      convergence = fit3$convergence, time = t3["elapsed"]
    )
  }
  
  # Comparação
  if (verbose) {
    cat("\n=== Comparação de Estratégias de Otimização ===\n")
    cat(sprintf("%-15s %-40s %-15s %-10s %-10s\n",
                "Método", "Parâmetros (alpha, beta, theta, rho)",
                "log-lik", "Conv.", "Tempo(s)"))
    cat(paste(rep("-", 90), collapse = ""), "\n")
    for (name in names(results)) {
      r <- results[[name]]
      cat(sprintf("%-15s (%.4f, %.4f, %.4f, %.4f) %15.4f %5d %10.4f\n",
                  name, r$par[1], r$par[2], r$par[3], r$par[4],
                  r$loglik, r$convergence, r$time))
    }
  }
  
  # Retornar o melhor resultado
  best <- which.max(sapply(results, function(r) r$loglik))
  return(list(best = results[[best]], all = results, best_method = names(results)[best]))
}
